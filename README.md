# [Human Resource Information System - Moraspirit Initiative](https://hris.moraspirit.com)

 ![version](https://img.shields.io/badge/version-1.0.0-blue.svg)  ![license](https://img.shields.io/badge/license-MIT-blue.svg) ![GitHub issues open](https://img.shields.io/github/issues/creativetimofficial/black-dashboard/issues.svg?maxAge=2592000)

<p align="center"><a href="https://www.moraspirit.com" target="_blank"><img src="https://github.com/moraspirit/msp-hr-website/blob/dev/public/black/img/msplogo1.jpg?raw=true" width="400"></a></p>

Moraspirit HRIS is the web application for managing Members, Recruitments, Member Performance, Issuing Certificates etc... It has been built using Laravel with MySQL as the database. Frontend has been built with HTML, CSS and JavaScript.
